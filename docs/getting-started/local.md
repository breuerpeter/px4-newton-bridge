# Local

Run PX4 SITL with the Newton simulator locally.

## Prerequisites

- [PX4 build dependencies](https://docs.px4.io/main/en/dev_setup/dev_env.html)
- [uv](https://docs.astral.sh/uv/)

## Usage

```bash
make px4_sitl_newton newton_astro_max
```

The bridge dependencies are installed automatically into a virtual environment on the first run.
