# Contributing

Thanks for your interest in contributing to PX4-Newton Bridge!

## Getting Started

- [Development setup](https://breuerpeter.github.io/px4-newton-bridge/contributing/development/)
- [Documentation](https://breuerpeter.github.io/px4-newton-bridge/contributing/docs/)

## Reporting Bugs

Open a [GitHub Issue](https://github.com/breuerpeter/px4-newton-bridge/issues). Include:
- Steps to reproduce
- Expected vs actual behavior
- Output of `nvidia-smi` (if GPU-related)

## Pull Requests

- Use [conventional commits](https://www.conventionalcommits.org/) with a scope: `type(scope): description`
- Install [pre-commit hooks](https://breuerpeter.github.io/px4-newton-bridge/contributing/development/#pre-commit-hooks) before committing
- PRs are validated by CI and [pre-commit.ci](https://pre-commit.ci)
